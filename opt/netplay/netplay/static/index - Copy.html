Quick Start

  • Open 4 tabs/windows; choose a name in each.
  • Connect seats: White, Grey, Black, Pink.
  • Decide Human vs AI for each seat.
  • When White shows GO (flashing), start playing.
  • Press New Game (and Confirm) to reset anytime; duel resets all four.
